# Kenya Price Comparison Tool

An AI-powered price comparison tool that scrapes multiple 
Kenyan shopping sites including Jumia and Kilimall, 
then uses AI to analyse and recommend the best value products.

## Features
- Scrapes product names, prices and links
- Supports multiple shopping sites
- AI powered analysis using local Ollama models
- Delivers results in clean text format

## Installation
pip install -r requirements.txt

## Usage
python jumia_seo.py

## Sites Supported
- Jumia Kenya
- Kilimall
